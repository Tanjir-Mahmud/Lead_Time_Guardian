# Lead-Time Guardian

A high-performance logistics auto-pilot app for the 2026 RMG and Pharma sectors.

## Features
- **Dashboard**: Real-time KPI monitoring.
- **Document Auditor**: AI-powered invoice and bill of entry analysis.
- **Logistics Agent**: Chat assistant with reasoning capabilities.
- **Smart Maps**: Live route tracking.

## Prerequisites
- Node.js 18+
- OpenRouter API Key (configured in `.env.local`)
- Supabase Project (configured in `.env.local`)

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install --legacy-peer-deps
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Open Application**
   Visit [http://localhost:3000](http://localhost:3000)

## Build
To build for production:
```bash
npm run build
```
